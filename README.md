## Basic framework for a complete website

* Automatic Object Persistence
* Account Management
  * User creation
  * Salted passwords
  * Roles
* OAuth 2 with providers model
  * Login with Google
* Automatic Component Loading
* Automatic Website Initialization
* Logging Component
